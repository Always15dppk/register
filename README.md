## 📋 What is register?

register is a collection of three tools designed to automate account registration. Each tool focuses on a specific service:

- **openai-register/**: Automatically creates an OpenAI account. See the README inside for details.
- **grok-register/**: Automates registering on Grok (x.ai). Check its README folder for guidance.
- **tavily-register/**: Helps you register an account on Tavily automatically. Instructions are inside the directory README.

These are scripts aimed at simplifying sign-ups where possible.

To run these tools, your Windows computer needs:

- Follow instructions to run the registration script.

Scripts generally start with a simple command like `.\register.ps1` or `python register.py`.
